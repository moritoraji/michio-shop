お客さんの「困った」を
チームで解決
していく面白さ

楽天市場のサイトを担当する同じチームの3名。
入社2年目の若手Nさんが、
基本的な受注や出荷の準備などをメインに担当し、
1つ上の先輩WさんがNさんのサポートをはじめ、
バナーや商品画像などを制作。
キャリアの長いKさんは、
売上を伸ばすために自社の企画を考えたり、競合他社の動向や分析、
新作商品の出品作業などを担当しています。
そんな普段からチームワークの
いいメンバーたちの本音を探ってみました。

MEMBER

K.T

インターネット事業部
2006年入社

岡山県総社市出身。芸術学部卒。大学生のときに、地域の会社と1冊の本を作るワークショップで、ミチオショップの代表・竹内氏と出会う。最近は生き物（昆虫・魚・爬虫類）が趣味になり、自宅の庭に専用の小屋を作り、空調なども完備。自身で販売なども行っている。

N.A

インターネット事業部
2022年入社

山口県下松市出身。服飾美術学科卒。短大進学で岡山に出てきて、倉敷の風土が気に入り、地元には戻らずに就職。ミチオショップを知ったときは、岡山にネット通販をしている会社があったんだ！と驚く。高校時代から美術部に所属していて絵は描くのも見るのも好き。

W.M

インターネット事業部
2021年入社

岡山県倉敷市出身。学生時代は美術部・放送文化部に所属。ネットラジオを運営したり、ラジオドラマを制作した経験があり、その美声でミチオショップの自動案内の電話音声も担当。プライベートではK-POPのライブに行ったり、友人とライブ鑑賞にハマっている。

THEME 01

お互いの印象について

  • K.T :

    ECサイトの運営って、パソコンだけで完結するようなイメージがあるけど、実はお客さんの問合せ電話に対応することも多くて、N.Aさんは入社2年目だけど、苦手意識を持たずにこなしている印象があります。

  • N.A :

    本当ですか！？その評価は自分的には信じられないくらい、もともとコミュニケーションをとるのが苦手なんですよ。もう1vv年目はあっぷあっぷだったんですけど、それこそW.Mさんが横目でチラチラ見てくれて（笑）。私が困っていそうだったら「どうしたん？」と声をかけてくれて本当に助かっています。

  • W.M :

    N.Aさんはコミュニケーションをとるのが苦手というか、不思議ちゃんかな。

  • N.A :

    それ、めっちゃ言われます！

  • W.M :

    いい意味で不思議ちゃんだから、注文依頼が立て込んでこっちに余裕がないときでも、「大丈夫、行けますよー」と独特の空気感を出してくるので、なんか心に余裕ができるというか、こっちも助かっています。

  • N.A :

    ありがとうございます。うちの社内って結構、年齢の近い先輩も多いんですけど、入社数年の差で人間の出来が違うというか。もうすでに歴戦の猛者感ありますよね。

  • W.M :

    いやいや、キミもいい性格しているよ。

THEME 02

メンバーのココを
見習いたい！

  • K.T :

    やっぱりN.Aさんの若手ならではの動きって新鮮な気持ちになることがあって。前にお祭りのときの白い足袋を探しているお客さんがいたよね？

  • W.M :

    ああ、あのときの！秋にお祭りが重なるから、どこも欠品していて、すごく困っているお客さんいました。その対応をしたのがN.Aさんで、困っているお客さんのためにいろんなメーカーさんの在庫を必死に問い合わせて、「ちょっと値段が上がるけど、このタイプならあります！」って。

  • N.A :

    いやぁ、それで結果的に売り上げも上がって…。

  • W.M :

    どうしても仕事に慣れちゃうと最初から諦めてしまうことがあるんだけど、何とかしようという一心で道を切り拓いていて、すごく感銘を受けました。

  • N.A :

    本当に必死なだけなんですけど、私は逆に先輩方の冷静に先を見通す力がすごいな、と思いますね。

  • K.T :

    というと？

  • N.A :

    例えば、台風が近づいてきているけど、この商品の日付・時間指定って間に合う？って指摘されたり、最悪のケースも想定したコミュニケーションをとっておかないといけないな、と気づいたり。

  • W.M :

    あとはメールのやり取りとかメルマガで文章力も鍛えられるよね。

  • N.A :

    そうですね！メルマガも私が作成した文章をみんなにチェックしてもらうんですけど、こういう書き方の方が伝わるよねとアドバイスをもらえて文章力も向上しますし、色々な人の意見を聞けること自体が社内のコミュニケーションにもなって、すごくありがたいです。

THEME 03

ECサイトならではの
面白さとは？

  • N.A :

    やはり実店舗と違って、全国にお客さんがいるので、沖縄や北海道のお客さんから注文があると、どんな生活しているのかなぁと想像したりするのが面白いです。

  • W.M :

    さっきのメルマガのチェックもそうだけど、実店舗の接客って対面での１対１だけど、ネット通販だとお客さんの要望に対して、こっちはチームで対応できるから、色々な視点でサポートできるし、じっくり調べたものを提案できるのはネットの強みだと思う。

  • K.T :

    しかもミチオショップって、お客さんの要望にどれだけ応えられるかを大事にしているから、他のECサイトよりもお客さんへの寄り添い方が違うというか。ワーク用品の分野って、他のアパレルと違って必需品なので、なくて困っているという人も多い。そこに「在庫がありません」と断るのではなくて、サイズ違いなら、色違いなら揃えられます、というところまで踏み込んで提案しているのは、ミチオショップの魅力だと思う。

  • W.M :

    その結果、名指しで注文してくれるお客さんもいたりして。

  • N.A :

    結構やりがいを感じる場面が多い仕事ですよね。
