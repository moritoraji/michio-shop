小さな壁を
越えていくことが
自分にとって
大きな一歩になる

ミチオショップは、前向きでポジティブな会社です。
失敗を恐れず、チャレンジし続けてきたからこそ、
時代に飲み込まれるのではなく、
その波を乗りこなしていく力づよさが生まれました。
そんな「チャレンジ」をキーワードに、
これからのミチオショップを支える若手社員3名による
クロストークを開催！
それぞれ部署の違う3名が、どのように成長を
遂げたのか、赤裸々に語ってもらいました。

MEMBER

K.M

総務部
2022年入社

岡山県岡山市出身。ブライダル学科卒。中学では吹奏楽部でクラリネットを担当、高校では放送部に所属。プライベートでは東京に上京した友人に呼ばれて月１回は東京へ。地下アイドルからロック系のフェスまで、アクティブにライブ参戦。取り柄は行動力とのこと。

N.A

インターネット事業部
2022年入社

山口県下松市出身。服飾美術学科卒。短大進学で岡山に出てきて、倉敷の風土が気に入り、地元には戻らずに就職。ミチオショップを知ったときは、岡山にネット通販をしている会社があったんだ！と驚く。高校時代から美術部に所属していて絵は描くのも見るのも好き。

I.R

物流事業部
2023年入社

岡山県倉敷市出身。メディア芸術学部卒。高校では服飾系を専攻していて、自分で服を作るのも好き。趣味は編み物とバレーボール。男子バレーが好きでVリーグを観戦したりも。最近では物流事業部のパートさんに誘われて、自身でもバレーのプレーを楽しんでいる。

THEME 01

これまで、どんな壁を
乗り越えてきた？

  • N.A :

    私はとにかくコミュニケーションをとるのが苦手だったので、最初は電話に出るのがイヤでイヤで。でも、それを克服するのは数をこなすしかない。先輩のトークを隣で聞きながら、「これでいいんだ」と言う経験を増やしていった感じかなぁ。

  • K.M :

    私は新卒でまさかの総務部配属だったんで、いや計算も苦手だし、社交性もないし、どうしようと思って（笑）。

  • N.A :

    K.Mさんは何でもソツなくこなしそうな感じだから、総務部だったんじゃないかな。

  • K.M :

    とにかく今でも毎日が勉強。自分用のノートを作って、わからないことは書きまくって。人には迷惑をかけないのが私のモットーなので。

  • N.A :

    そういうところが総務に向いているんだと思う（笑）。

  • K.M :

    でも、２年目になって採用に関わらせてもらえるようになって、ちょっと嬉しかったなぁ。

  • I.R :

    私はインスタの担当になったことがチャレンジのひとつ。もともとSNSはあんまりやらないタイプだったので、他のアパレルさんの投稿を参考にしたり、リール作った方がいいんだなぁと勉強して、パートさんに踊ってもらったり。

  • K.M :

    あれ、めちゃくちゃ可愛かった！

  • I.R :

    みんなノリノリでやってくれてありがたかった。ああいう自信作のときは、みんなに「見て！」って声をかけています。

THEME 02

この人スゴイ！
とリスペクトしたことは？

  • N.A :

    I.Rさんの物流事業部の工夫がすごいなぁと思って。

  • I.R :

    なんだろう？

  • N.A :

    倉庫に大量の商品が入荷したとき、同じメーカーさんの段ボールが並ぶと、どこになんの商品が入っているのか分かりづらくて…それを分かりやすく区分けして品番を書いてくれて、めちゃくちゃ助かっています。

  • I.R :

    あれは同じアイテムでも全サイズの商品が同じ段ボールに入っていたから、一度全部バラして、並べ直して、分かりやすく記載するようにして。

  • K.M :

    あのおかげで、この商品どこで見た？あっちで見た気がする、みたいに右往左往することがなくなったよね。

  • N.A :

    今は品番さえ叫べば、誰かが「ココだよ！」と教えてくれて（笑）、めちゃくちゃスムーズになった。

  • K.M :

    私がスゴイなぁと思うのは、同じ総務部の先輩。自分から中小企業向けのこんな制度が使えそうだと調べて、しっかり会社に利益をもたらして、シンプルに尊敬できる。尊い！

  • I.R :

    そんなに年が離れているわけではないんだけど、先輩たちは本当にリスペクトです。物流の私の上司と総務のK.Mさんの先輩と外商の先輩3名で、リクルート向けの社内アンケートを作っていたんですけど、通常業務にプラスして全従業員に声がけしたりしていて、すごいなぁと。

THEME 03

今後チャレンジ
していきたいこと

  • K.M :

    慣れない総務でも徐々にやれることが増えてきたので、大きな銀行さんとのやり取りも任せてもらえるように頑張っていきたいですね。

  • I.R :

    物流事業部は、繁忙期の夏に短期のアルバイトさんを採用するんですが、その指導を課長に任せっきりになっているので、自分が教育できるようになりたいですね。課長と私の2人で、パートさんやアルバイトさんをまとめているので、しっかりマネジメントできるように成長したいです。

  • N.A :

    私はまず基本の受注対応をスピーディーにできるようになって、新しい商品の出品を任せてもらえるように。注文の対応に追われていると、商品の説明文だったり、バナーや商品画像の制作に手が回らないので、とにかく基本業務をスムーズにできるようになることが当面の目標です。

  • K.M :

    それにしても、気がついたら一人で仕事をするのも怖くなくなってたんだよね。

  • N.A :

    それはあるね。とにかくフレンドリーな会社だから、周りのサポートもあって、安心して成長できるというか。

  • K.M :

    理不尽なことを言う人もいないし、先輩に何を話しかけても大丈夫だし。

  • I.R :

    新卒社員は物流での研修からスタートしますけど、おそらく一度は誰もがミスするもの。何でこんなことで…とヘコむと思うんだけど、一回ミスしたくらいでは誰も怒らないので。

  • N.A :

    コミュニケーションが得意じゃなかった私ですら、これだけできるようになったんだから、人としても成長できる職場じゃないかな。
